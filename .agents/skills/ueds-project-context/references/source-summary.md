# Resumen del anteproyecto

El anteproyecto describe una máquina de extracción mecánica de sacarosa mediante rodillos, motor eléctrico, transmisión, estructura metálica, canal de recolección y elementos de seguridad. Sus ejes narrativos son eficiencia del aprovechamiento, reducción del esfuerzo físico, seguridad industrial, valor académico y aprovechamiento del bagazo.

Usar este resumen solo como orientación. El contenido oficial del proyecto para la web está consolidado en `docs/PROJECT_CONTEXT.md`.
